__DATA__ - データセクション

__DATA__より下に書かれたものは、データセクションと呼ばれます。データセクションを利用すると入力ファイルの内容を、プログラムの中に記述することができます。

DATAファイルハンドル

データセクションに記述された内容は、DATAという特殊なファイルハンドルを使って読み込むことができます。 

while (my $line = ) {
  print "$line";
}

__DATA__とDATAファイルハンドルのサンプル
