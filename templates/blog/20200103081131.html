請求書の作成 - PDF::API2で帳票作成

PDF::API2で請求書を作成してみました。見積書、申込書、レシート、サービス利用明細など、本格的な帳票の作成の基本的なひな型として利用できます。
  • A4用紙サイズ
  • 日本語対応
  • 太字対応
  • 背景色対応
  • フォントサイズ対応
  • 下線対応
  • 表の表示
  • 表の偶奇カラーリング
  • 単価、数量、価格
  • 価格合計
  • 印鑑押印用の四角
  • 会社ロゴの挿入
  • 金額桁区切り
  • 25項目

PDF::API2で書いた請求書のソースコード

PDF::API2で書いた請求書のソースコードです。 会社ロゴを入れる場合は、同一ディレクトリに、会社ロゴlogo.pngを入れて、ソースコードの該当箇所のコメントアウトをはずしてください。 
use strict;
use warnings;
use utf8;
use FindBin;

use PDF::API2;

my $books = [
  {
    name => '本1',
    unit_price => 1000,
    count => 3,
  },
  {
    name => '本2',
    unit_price => 2000,
    count => 6,
  },
  {
    name => '本3',
    unit_price => 1500,
    count => 5,
  }
];

my $total_price_no_tax = 0;
for my $book (@$books) {
  $total_price_no_tax += $book->{unit_price} * $book->{count};
}
my $total_price_no_tax_disp = price_disp($total_price_no_tax);
my $tax_rate = 0.1;
my $tax = int($total_price_no_tax * $tax_rate);
my $tax_disp = price_disp($tax);
my $total_price = $total_price_no_tax + $tax;
my $total_price_disp = price_disp($total_price);

my $pdf = PDF::API2->new;

$pdf->mediabox('A4');

my @page_size_infos = $pdf->mediabox;
my $page_width = $page_size_infos[2];
my $page_height = $page_size_infos[3];

my $default_font_size = 10;

my $page_top_margin = 48;

my $render_start_x = $page_left_margin;

# 描画終了x座標
my $render_end_x = $page_width - 1 - $page_right_margin;

# 描画終了y座標

# 日本語に対応したTrueTypeフォントの読み込み

my $font_bold = $pdf->ttfont($true_type_font_bold_file);

my $cur_row = 0;

$gfx->linewidth($line_width_basic);
$gfx->strokecolor('#000');
$gfx->stroke;
